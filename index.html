How to get configs / executor (iOS only)

How to get Roblox Executor (iOS)

  1. Go to this link and tap Allow.
  2. Go to Settings > General > VPN & Device Management. You should see koindns under 'Downloaded Profile'. Tap it, then tap Install and enter your passcode (if prompted). Tap Install again.
  3. Go back to VPN & Device Management and tap on DNS (under VPN Configuration). Ensure both Adware Filter 1 and Adware Filter 2 are selected (have checkmarks). Switch from Automatic to one of these if needed.
  4. Go to your home screen. You should find a new app called esign khoindvn. Open it.
  5. Scroll down until you see esign bypass revoke. Tap on the text below it (e.g., avex digital INC or similar - this is your certificate name).
  6. Attempt to open the 'esign' app that appeared. If it shows an 'Untrusted Developer' message noting your certificate name, go to Settings > General > VPN & Device Management. Under 'Enterprise App', tap your certificate's name, then tap Trust "[Certificate Name]", and finally tap Trust again. (You might need to restart your device).
  7. After trusting, open the esign app again and agree to any prompts (like notifications).
  8. Go back to the esign khoindvn app and tap Download esign certificate. Tap More... and choose Save to Files. Save it somewhere accessible.
  9. In the main 'esign' app (the one you trusted), tap the Settings tab (bottom right), then Import Resource File. Find the `.zip` file you just saved (e.g., `esign_certs.zip`), tap it, then tap Unzip.
  10. After unzipping, navigate into the newly created folder (e.g., esign certs). Find the file matching your certificate name (it will likely end in `.p12`). Tap it, select Import Certificate Management, enter `1` as the password if prompted, and tap OK. It should say Imported Successfully. Tap the back arrow/done.
  11. Go to this link to download Delta IPA. In Safari, tap the downloads icon (down arrow), find the file, tap it. Then tap the Share icon (square with up arrow) and choose Save to Files. Save it.
  12. Open the 'esign' app again. Go to the Apps tab (bottom left). Tap Import (top right). Find the Delta...ipa file you saved, tap it. If it asks to Import to App Library, tap No. Wait for it to import.
  13. Once imported, find Delta in the app list. Tap on it, then tap Signature. Ensure Install After Signed is turned ON. Tap Signature again and wait for the process.
  14. A prompt should appear asking to install "Delta". Tap Install. If multiple prompts appear, tap Cancel on the extras. Check your home screen - Delta should be installing!

Note: This process involves sideloading and certificates which can be complex. If you need help with specific steps like getting the Delta key after installation, searching on YouTube or relevant communities is recommended.
